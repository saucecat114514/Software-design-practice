# 高级软件设计实践 · 教育培训机构教务收费管理系统

软件工程专业大三实训项目：4 周走通 **AI 赋能的全流程软件工程**闭环 —— 需求获取 → 需求分析 → 需求规格说明 → 验证确认 → 架构选型 → 约束设计 → AI 代码生成(v1) → 需求变更管理 → 代码变更(v2) → 逆向校验 → 设计资产归档。

- **选定案例**：教育培训机构教务收费管理系统（5 类涉众、35 个功能模块）。
- **周期**：2026-06-15 ~ 2026-07-10（20 个工作日）。
- **技术栈**：Java + Spring（Controller→Service→Repository→Model 四层）。
