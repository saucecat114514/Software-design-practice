# wiki/baselines —— 第 3 层 · 基线版本层

存放经 CCB 正式评审审批的**「不可变」版本**。需求/设计冻结后复制进来。

- 基线目录命名：`BL-YYYYMMDD-NN`（NN = 当日流水号）
- 典型结构：
  ```
  BL-YYYYMMDD-NN/
  ├── SRS-正式版.md  需求清单.md  UML模型/  溯源矩阵.md
  ├── adr/ style/ module/ topology/ constraint/ oas/   # 设计产物
  └── change/ report/                                   # 变更三剑客 + 校验报告
  ```
- 铁律：基线文件**一旦创立不得修改、不得删除**；需求变更只**新建**基线目录，旧基线完整保留。

详见 [[00_项目总作战计划]]。
